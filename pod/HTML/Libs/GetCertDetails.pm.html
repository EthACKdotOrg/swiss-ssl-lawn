NAME

GetCertDetails - Get certificate details

SYNOPSIS

use GetCertDetails qw(check_protocol_cipher);

$cert_details = get_cert_details( $host, $port );

DESCRIPTION

This module get the host certificate and extract all informations.

Extracted data

  • Subject
  • Issuer
  • Alternative names
  • Ashes/fingerprints
  • Expiration date
  • Serial number
  • Version
  • Extensions (oid, nid, ln, sn, data)
  • CRL distribution points
  • Extended key usage
  • Netscape cert type
  • Certificate, signature and public key info
  • Certificate in .pem format
  • OCSP info and OCSP validation

Arguments

  • $host # The host name
  • $port # The host port

Return

  • return $cert; # Hash ref with all certificate inforation
